Projeto para a matéria de linguagem de programação aplicada do curso de ADS da Uninter

Esta é uma demo de um roguelite 2d onde uma pessoa naufragada tem que sobreviver a hordas de inimigos e pescar upgrades entre elas.
  -O layout do background do jogo foi desenhado a mão e então processado pelo chat gpt para ser minimamente agradavel.
  -O restante dos sprites e imagens foram todas ou feitas do zero ou edições de projetos gratuitos
    -programs usados: Pycharm, Affinity, [Pixilart](https://www.pixilart.com/draw) e [AudioMass](https://audiomass.co/)
    -characters template pack by ErisEsra [itch.io](https://erisesra.itch.io/character-templates-pack)
  -Os SFX e a musica foram adquirido através dos sites Uppbeat e Pixabay
